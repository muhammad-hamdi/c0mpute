# transcode — web surface (stub)

Future home of the transcode plugin's web dashboard, intended to mount at
`c0mpute.com/transcode` once we build it. Held for v1; per-plugin web /
PWA / desktop surfaces are deferred (see README at the repo root).

For v1, the only web surface is the c0mpute landing at `apps/web`, which
explains the CLI install. Everything else — uploading videos, viewing
jobs, etc. — happens through `c0mpute transcode <command>`.
